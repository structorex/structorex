# Structorex Dokumentation

Willkommen bei **Structorex**, dem ultimativen Kommandozeilen-Tool zur Visualisierung von Projektstrukturen und zum Export von Quellcode-Kontexten.

Structorex macht es unglaublich einfach, Code einzulesen, unwichtige Dateien zu ignorieren und umfassende Berichte in Markdown, JSON, HTML oder Text zu erstellen. Besonders nützlich, um KI-Modellen (LLMs) Kontext zu geben oder saubere Backups zu ziehen.

## Kernfunktionen
- 🌳 Visuelle Baumdarstellung
- 📄 Vollständige Einbettung des Codes
- 🚦 Integrierte `.gitignore` und Security-Filter
- 📦 ZIP Snapshot Erstellung
- 🔣 Automatische Erkennung der Textcodierung
