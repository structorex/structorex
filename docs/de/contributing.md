# Mitwirken

Beiträge sind immer willkommen! Egal ob Bug-Reports, Feature-Vorschläge oder Pull Requests.

## Entwicklungsumgebung

1. Forke das Repo.
2. Abhängigkeiten installieren:
   ```bash
   pip install -e .[dev]
   ```
3. Tests ausführen:
   ```bash
   pytest tests/
   ```

## Richtlinien
- Halte dich an den **PEP 8** Standard.
- Stelle sicher, dass Tests erfolgreich sind.
- Schreibe verständliche Commit-Nachrichten.
