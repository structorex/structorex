# Installation

Structorex ist in Python geschrieben und über PyPI verfügbar. Benötigt Python 3.8+.

## Globale Installation (Empfohlen)

Ganz einfach über pip installieren:

```bash
pip install structorex
```

## Aus Quellcode

```bash
git clone https://github.com/yourusername/structorex.git
cd structorex
pip install -e .
```
