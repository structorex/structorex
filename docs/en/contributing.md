# Contributing

We love contributions! Whether it's reporting a bug, proposing new features, or submitting a Pull Request, all help is appreciated.

## Setting up for development

1. Fork the repo.
2. Install dependencies:
   ```bash
   pip install -e .[dev]
   ```
3. Run tests before committing:
   ```bash
   pytest tests/
   ```

## Guidelines
- Follow **PEP 8** style.
- Ensure tests pass and add coverage for new features.
- Write descriptive commit messages.
