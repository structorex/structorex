# Structorex Documentation

Welcome to **Structorex**, the ultimate command-line tool for visualizing project structures and exporting code contexts.

Structorex makes it incredibly easy to scan your source code, ignore irrelevant files, and generate comprehensive reports in Markdown, JSON, HTML, or raw text. It's particularly useful for providing project context to Large Language Models (LLMs) or for archiving your codebase cleanly.

## Key Features
- 🌳 Visual tree representations
- 📄 Complete file content embedding
- 🚦 Built-in `.gitignore` and security filtering
- 📦 ZIP Snapshot creation
- 🔣 Automatic text encoding detection
