# Документация Structorex

Добро пожаловать в **Structorex**, лучший инструмент командной строки для визуализации структур проектов и экспорта контекста исходного кода.

Structorex невероятно упрощает сканирование кода, игнорирование ненужных файлов и создание подробных отчетов в форматах Markdown, JSON, HTML или Text. Это особенно полезно для передачи контекста моделям ИИ (LLM) или для создания чистых резервных копий.

## Ключевые возможности
- 🌳 Визуальные древовидные структуры
- 📄 Полное встраивание содержимого файлов
- 🚦 Встроенная поддержка `.gitignore` и фильтры безопасности
- 📦 Создание ZIP-снимков
- 🔣 Автоматическое определение кодировки текста
