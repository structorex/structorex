# Участие в разработке

Мы всегда рады помощи! Будь то отчет об ошибке, предложение новой функции или Pull Request.

## Среда разработки

1. Сделайте Fork репозитория.
2. Установите зависимости:
   ```bash
   pip install -e .[dev]
   ```
3. Запускайте тесты перед коммитом:
   ```bash
   pytest tests/
   ```

## Руководство
- Следуйте стандарту **PEP 8**.
- Убедитесь, что тесты проходят.
- Пишите понятные сообщения коммитов.
