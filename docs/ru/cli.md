# Справочник по CLI

Обзор всех доступных аргументов командной строки.

```text
usage: structorex [-h] [-o OUTPUT] [-f {text,json,markdown,html}] [-e [EXCLUDE ...]] [-a [ALLOWED ...]] [-m MAX_SIZE] [-v] [-s] [-z EXPORT_ZIP] [path]
```

### Аргументы
- `path` (Опционально): Каталог для сканирования. По умолчанию - текущий каталог (`.`).
- `-o, --output`: Имя выходного файла. По умолчанию `project_report.txt`.
- `-f, --format`: Формат вывода. Доступно: `text`, `markdown`, `json`, `html`.
- `-e, --exclude`: Дополнительные исключаемые папки (например, `docs` `tests`).
- `-a, --allowed`: Конкретные разрешенные расширения (например, `.py` `.txt`).
- `-m, --max-size`: Максимальный размер файла в МБ. По умолчанию: 10.
- `-v, --verbose`: Включить подробное логирование.
- `-s, --show-skipped`: Показывать пропущенные файлы в дереве.
- `-z, --export-zip`: Создать ZIP-архив вместе с отчетом.
