# Установка

Structorex написан на Python и доступен на PyPI. Требуется Python 3.8+.

## Глобальная установка (Рекомендуется)

Установите легко с помощью pip:

```bash
pip install structorex
```

## Из исходного кода

```bash
git clone https://github.com/yourusername/structorex.git
cd structorex
pip install -e .
```
